# DayDesign

极简的组件库。

## Why

大多数用于 Web UI 的组件库，都追求简洁高效的风格。这对于一般的 Web App 来说很好，但不是我的博客想要传达的感受。
因此，在写我自己的博客前端 Daydreamer 的时候，我选择不使用现有的组件库，用 Tailwind 自己设计所有组件。
而这也带来了样式耦合、难复用等问题。

所以，我将 Daydreamer 前端里的组件做成了组件库，叫做 DayDesign。
